✥ Unit 1 - Introduction to Machine Learning (ML)

Outcomes from the Team Exercises and activites

Correlation signifies the strength and direction of a linear relationship between two variables, offering insights into feature interdependencies, a pivotal aspect in feature engineering and selection. On the other hand, regression provides a framework to predict the output of a variable based on one or more inputs, making it central to predictive modelling in machine learning (Rizwan, 2020).

A way to assess the direction and strength of a linear relationship between two quantitative variables is through correlation. The range of the value is from -1 to 1. A score around 1 denotes a significant positive correlation, meaning that as one variable rises, so does the likelihood that the other will as well. As one variable increases, the other tends to decrease, therefore a number close to -1 denotes a high negative connection (Nair et al, 2023).

It is used in machine learning for:

  • Feature Selection: Correlation matrices are often used to select relevant features. If two features are highly correlated, they carry redundant information. One of them can be dropped to reduce dimensionality.
  • Detecting Multicollinearity: In linear regression models, multicollinearity can distort the prediction. Correlation analysis helps in detecting such scenarios.
  • Understanding Relationships: Before building complex models, correlation analysis gives a quick overview of relationships between variables.

Regression is used to model and analyse relationships between variables. It offers a framework for forecasting output based on one or more inputs. It is extensively used for prediction and prognostication, and its use has substantial overlap with the domain of machine learning (Bansal et al 2021).

Types:

  • Linear Regression: Assumes that the dependent and independent variables have a linear relationship.
  • Logistic Regression: This method is used to solve binary classification problems.
  • Polynomial Regression: This method captures non-linear correlations by using polynomial characteristics.

It is used in machine learning for:

  • Predictive Modelling: The primary application is forecasting where future values are predicted based on historical data.
  • Understanding Feature Importance: The significance of features in formulating predictions can be revealed by regression coefficients.
  • Evaluating Trends: Regression models can detect and highlight underlying trends in data.

Reflection & Practical Use:

Machine learning involves understanding data and statistics, loading data, exploring its characteristics, choosing a model, training the model, and evaluating its performance. The choice of model depends on the data's nature and variable relationships. Striking a balance is crucial, as polynomial regression can fit non-linear data better but can overfit if too high. Regularly evaluating model performance and understanding metrics like R2 score is essential. In the context of this unit, we traversed various aspects of machine learning, from data handling to model selection, fitting, and evaluation. The tools and methodologies used here are foundational to many machine learning tasks.
